# 偏差値計算アプリ

### 機能
テストを受けた全員の点数を入力して偏差値を計算できます！

### 操作方法
- テストを受けた人の点数を「点数」の列に入力すると「偏差値」の列に偏差値が計算されます
- 「+」「-」ボタンを押すとテストを受けた人を追加・削除できます
- テストを受けた人の名前は編集できます
- 「リセット」ボタンを押すと最初の入力状態に戻ります
